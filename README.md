# 🌾 Raitha Varta

Raitha Varta is an Android application developed to help farmers get short, simple, and daily crop advisories in an easy-to-read format.

## 📌 Problem Statement
Agricultural experts release long PDF advisories every week, but many farmers need quick and actionable information specific to their crops.

Raitha Varta converts advisory information into short cards with crop-wise updates.

## ✨ Features
- Daily crop advisory updates
- Kannada and English language toggle
- Crop-wise filter options
  - Paddy
  - Arecanut
  - Coconut
  - Tomato
- Swipeable advisory cards using ViewPager2
- Crop images
- Ask Expert button
- Daily date and day display

## 🛠️ Tech Stack
- Android Studio
- Java
- XML
- Material Design Components
- ViewPager2

## 📱 Screens
- Home screen with crop filters
- Advisory cards
- Expert assistance button

## 🚀 How to Run
1. Clone this repository
2. Open project in Android Studio
3. Sync Gradle
4. Run on emulator or Android device
